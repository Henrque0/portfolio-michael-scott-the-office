Este projeto é um portfólio fictício inspirado em Michael Scott, o icônico gerente regional da filial de Scranton da Dunder Mifflin na série *The Office*. 
	
Na série, Michael Scott deixa a empresa no episódio **“Adeus, Michael” (Temporada 7, Episódio 22)** para se mudar para o Colorado com sua noiva, Holly Flax, priorizando a vida pessoal e a família. 
Ele retorna apenas no final da série, na Temporada 9, durante o casamento de Dwight.

Com base nisso, surgiu a ideia deste projeto: imaginar como seria o portfólio profissional do **“Melhor Chefe do Mundo”** caso ele estivesse procurando um novo emprego.
Além disso, o projeto também serve como uma ótima desculpa para reassistir a série e atualizar as frases do personagem para que façam sentido dentro de um portfólio online.

As tecnologias utilizadas neste projeto foram **HTML e CSS**, e ele foi desenvolvido com fins de estudo e prática, inspirado no universo da série *The Office*.
